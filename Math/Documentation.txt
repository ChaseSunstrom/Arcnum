rollDice:
    Roll dice takes two unsigned int as arguments.
    Returns a random unsigned int between the min and max.
    Example: unsigned int roll = rollDice(1, 6); 
    1 <= roll <= 6
multiDiceRoll:
    multiDiceRoll rolls multiple dice to return an array of rolled ints.
    This takes 3 arguments (all unsigned int): min, max, number of dice to roll.
    Example: multiDiceRoll(3, 100, 5);
    returns {4, 12, 36, 19, 89}
